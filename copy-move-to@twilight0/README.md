# Copy/Move To Action (Nemo/Dory)

A custom context menu action for Nemo and Dory file managers to copy or move selected files and folders.

## Features
- Centered folder picker dialog.
- Persists the last destination directory.
- Falls back to the user's home directory.
- Avoids filename conflicts by appending a numbered suffix.
- Fully compatible with GTK 3 / Wayland desktop portal standards using `Gtk.FileChooserNative`.
