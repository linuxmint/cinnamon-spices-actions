# Windsurf-Launcher

Launch Windsurf with the current directory added to the workspace.

## DESCRIPTION

This is a cinnamon action that adds a menu entry to launch windsurf in the CWD.

## DEPENDENCIES

- `windsurf` install it from this link: https://windsurf.com/download/editor

## USAGE

Right-click in Nemo within a folder → choose ‘Open in windsurf’.
