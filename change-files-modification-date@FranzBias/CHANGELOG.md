## [1.0] - 2026-07-02

### Added
- optimised to be added to Nemo’s official actions

## [1.0] - 2025-08-12
### Added
- Initial release.
- Change the modification date (and time) of one or more selected files via a right-click context menu entry in Nemo.
- Optional backup of modified files (`.bkp` extension).
- Optional debug log of the changes made.
- Translations: English, Italian, German, French, Spanish, Portuguese.
